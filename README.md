# ClosureTools-VSCode

## Usage

1. cd server
2. npm install
3. code .
* [Open Server VS Code Window]
* execute Ctrl+Shift+B
4. cd ../client
5. npm install
6. code .
* [Open Client VS Code Window]
* execute F5
