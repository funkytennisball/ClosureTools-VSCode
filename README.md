# ClosureTools-VSCode

## Usage

1. cd server
2. npm install
3. code .
3.1. [Open Server VS Code Window]
3.2. execute Ctrl+Shift+B
4. cd ../client
5. npm install
6. code .
6.1. [Open Client VS Code Window]
6.2. execute F5
